# TOTP V3 Authenticator

一个精美的、基于浏览器的**动态口令（TOTP）验证器**。无需安装任何应用，打开网页即可生成双重验证码。

## ✨ 特性

- **纯前端实现**：所有计算在浏览器本地完成，密钥不上传服务器
- **现代化 UI**：基于 Tailwind CSS 构建，支持深色/浅色模式自适应
- **实时倒计时圆环**：可视化显示验证码刷新周期（30秒）
- **隐私保护模式**：模糊隐藏验证码，防偷窥
- **一键复制**：点击验证码即可复制，支持粘贴板操作
- **本地加密存储**：密钥在 LocalStorage 中混淆存储
- **URL 参数支持**：可通过 `?secret=` 或 `?key=` 参数快速加载密钥

## 🚀 快速使用

### 方式一：直接打开

1. 下载 `TOTP V3.html` 文件
2. 在浏览器中打开
3. 输入你的 Base32 密钥
4. 即可获取实时动态验证码

### 方式二：URL 参数

在 URL 中添加密钥参数快速加载：

```
index.html?secret=JBSWY3DPEHPK3PXP
```

或

```
index.html?key=JBSWY3DPEHPK3PXP
```

- **Google Fonts** - JetBrains Mono & Inter 字体

## 📝 核心算法

实现标准的 RFC 6238 TOTP 算法：

1. **Base32 解码**：将输入密钥解码为字节数组
2. **时间计数器**：计算当前时间步（30秒间隔）
3. **HMAC-SHA1**：使用密钥对时间计数器进行签名
4. **动态截断**：从 HMAC 结果中提取 4 字节，生成 6 位数字

## 🔒 隐私安全

- 所有计算在**浏览器本地**完成
- 密钥仅存储在**本地 LocalStorage**（混淆处理）
- 支持**隐私模式**，模糊显示验证码
- **无网络请求**，完全离线可用

## ⚙️ 配置

### 颜色主题

- 自动跟随系统深色/浅色模式
- 品牌色：`#3b82f6` (蓝色)

### 倒计时阈值

- 剩余 ≤ 5秒：红色
- 剩余 ≤ 10秒：黄色
- 其他：蓝色

## 🛠️ 开发说明

本项目为单 HTML 文件，所有依赖通过 CDN 加载。如需离线使用：

1. 下载所有 CDN 资源到本地
2. 修改 `script` 和 `link` 标签的引用路径
3. 或者直接将依赖内联到 HTML 中

## 📌 注意事项

- 请确保密钥为**有效的 Base32** 格式
- 验证码每 **30 秒** 刷新一次
- 时间同步：依赖客户端时间，请确保系统时间准确
- 支持 **Google Authenticator** 兼容的密钥格式
